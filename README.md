# Pruebas de la libreria GLIDE

## Documentacion

[Glide Docs](https://glidejs.com/docs/)

## Funcionamiento

### Dependencias

```console
npm install
```

### Comandos disponibles

```console
// Ejecutar en modo desarrollo
npm run dev
```

```console
// Ejecutar en modo produccion
npm run dev
```

#### Watch

```console
// Ejecutar en modo desarrollo
npm run dev-watch
```

```console
// Ejecutar en modo produccion
npm run dev-watch
```
